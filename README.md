# Weather App

A beautiful weather dashboard built with React and OpenWeatherMap API.

## Setup

1. Clone this repository
2. Run `npm install`
3. Copy `.env.example` to `.env` and add your OpenWeatherMap API key
4. Run `npm run dev`

## Environment Variables

You need to sign up at [OpenWeatherMap](https://openweathermap.org/) and subscribe to the One Call API 3.0 (includes 1000 free calls/day).

```
VITE_OPENWEATHER_API_KEY=your_api_key_here
```

## Features

- Current weather display
- 48-hour hourly forecast
- 8-day daily forecast
- Location search with geocoding
- Dynamic backgrounds based on weather conditions
- Responsive design with glassmorphism effects
